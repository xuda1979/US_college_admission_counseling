# Prompt Templates

## Holistic Evaluation Prompt
- Summarises applicant academics, extracurriculars, and goals.
- Requests JSON response with scores, narrative summary, and prioritized suggestions containing impact and effort estimates.

## Recommendation Prompt
- Uses latest evaluation summary and optional context to request three actionable recommendations.
- Asks for impact, effort, and deadline fields to support prioritisation and milestone planning.

## Essay Critique Prompt
- Frames the model as an admissions essay coach.
- Returns structured feedback with strengths, areas for improvement, and a revision plan aligned to the given prompt.
